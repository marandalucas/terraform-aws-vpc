# Description

This stack creates a vpc using the reference the module https://github.com/terraform-aws-modules/terraform-aws-vpc
This stack can create:
 - VPC
 - Public Subnet
 - Private Subnet
 - Internet Gateway
 - Publi—Å routes
 - Private routes
 - NAT Gateway
**/
